Tools

A collection of tools I’ve built over the years — some run right in your browser, some live on GitHub as source and releases, and the older ones are downloadable Windows utilities, each with the post that explains it.

Web toolsrun in browser

Bosch BES3 Reader

Read a Bosch BES3 e-bike drive unit over USB-C or Bluetooth and decode the full picture — model, battery health, drive-unit firmware, odometer and per-assist-mode usage — right in the browser.

Open tool Source

Marstek Venus Monitor

Advanced Web Bluetooth monitoring and diagnostics for Marstek Venus batteries — connect over BLE to read runtime, BMS, power and network data and run command tests, right in the browser.

Open tool Source

Marstek Firmware Analyzer

Extract certificates and security information from a Marstek firmware .bin image — all analysis runs locally in the browser, nothing is uploaded.

Open tool Source

Marstek Firmware Query Tool

Check which firmware versions Marstek's servers currently offer for your registered devices — a work-in-progress companion that queries the official update service.
